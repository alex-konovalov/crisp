4.5 Attributes and operations for formations

In addition to those available for generic group classes, formations also admit the following attributes and operations. See also Attributes and operations for Schunck classes for additional operations for saturated formations.

  • Residual(grp, form) O

    returns the form-residual of the group grp, i. e., the smallest normal subgroup res of grp such that grp/res belongs to form. Note that, unless form has an attribute ResidualFunction, there are presently only methods available for finite solvable groups.

  • ResidualFunction(form) A

    This attribute stores a function which computes the form-residual of a given group. In general, such a residual only exists if form is residually closed; cf. IsResiduallyClosed.

  • LocalDefinitionFunction(form) A

    For a saturated formation form with local function f, this attribute stores a function func taking two parameters, a group G and a prime p. If p is in the characteristic of form, this function must return a list list of elements of G, such that the smallest normal subgroup of G containing list is the f(p)-residual of G. If p is not in the characteristic of form, then func(G, p) must return fail for any group G.
