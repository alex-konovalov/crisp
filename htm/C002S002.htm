2.2 Properties of classes

  • IsEmpty(C)

    This property may be set to true or false if the class C is empty resp. not empty.

  • MemberFunction(C) A

    This attribute, if bound, stores a function which decides if an object belongs to C or not, and returns true and false accordingly. If present, this function is called by the default method for \in. MemberFunction should not be called directly by the user.
